# FamilyPet
The Ortega and Engmann households family pet


# Installation

```bash
pip install python-firebase
pip install requests
python configGenerator.py
```

Generate a config file for the drive train.
```bash
python configGenerator.py
```

# Running
```bash
python3 atom_controller_interface.py
```
